# Calculator App by [Pranit](https://pranitmane.com)

## Features
- Addition  
- Subtraction  
- Multiplication  
- Division  
- Percentage  

## Technologies Used
- React Native  
- Expo  
- Nativewind  

## Installation
1. Clone the repository  
2. Run `npm install`  
3. Run `npm start`
